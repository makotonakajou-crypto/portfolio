# 💪 筋トレ記録アプリ（Portfolio）

## 📌 概要

トレーニング内容を記録し、データを保存・可視化できるWebアプリです。
ユーザーが日々の成長を確認できるよう設計しました。

---

## 🔧 使用技術

* HTML / CSS
* JavaScript
* localStorage（データ保存）
* Chart.js（グラフ表示）

---

## ✨ 実装機能

* 種目・重量・回数の記録
* データの保存（ブラウザ保持）
* 日付付き記録
* 削除機能
* グラフによる成長の可視化
* Enterキーで入力可能
* 入力後のフォームリセット

---

## 💡 工夫した点

ユーザーの使いやすさを意識し、以下を実装しました。

* 入力後にフォームを自動リセット
* Enterキーでスムーズに追加
* データをグラフ化し、視覚的に成長が分かる設計
